# Danny Pham's Personal Website

This is a professional personal resume built with a focus on high performance, clean aesthetics, and interactive user experience. It highlights my education and professional experience.

### Core Framework & UI
* **HTML5 & CSS3**: Semantic structure for accessibility and custom styling for a polished look.
* **Simple.css**: A lightweight CSS framework providing a clean, readable foundation.
* **Font Awesome (v7.0.1)**: High-quality vector icons used for navigation and professional section headings.

### Interactive Elements
* **Vanta.js (Clouds)**: A dynamic, WebGL-powered 3D background that responds to mouse and touch input.
* **Custom JavaScript Engine**:
    * **Optimized Theme Switcher**: Features a lag-free light/dark mode transition using `.setOptions()` to update Vanta.js colors instantly.
    * **Typewriter Effect**: An automated typing animation for the header sub-headline.
    * **Scroll Progress Bar**: A fixed indicator at the top of the viewport tracking reading progress.
    * **Intersection Observer**: Custom "scroll-reveal" animations that trigger as elements enter the viewport.

### Design & Performance
* **Responsive Design**: Fully mobile-friendly layout with media queries for consistent experiences on all screen sizes.
* **Performance Focused**: Efficient DOM manipulation and optimized resize handlers to ensure smooth background animations.
